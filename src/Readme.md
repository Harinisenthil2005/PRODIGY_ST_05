# Task 05 - E-Commerce Checkout Automation

## Project Title
Shoplane E-Commerce UI Automation Testing

## Description
This project automates the checkout flow of an e-commerce website using Selenium WebDriver with Java. It validates the end-to-end functionality of the application including product search, cart operations, and checkout process.

## Tools & Technologies
- Java
- Selenium WebDriver
- ChromeDriver
- VS Code / Eclipse
- TestNG (optional)

## Project Structure
- src/ → Automation scripts
- TestCases.xlsx → Test scenarios
- BugReport.docx → Bug reports

## How to Run
1. Install Java and Selenium
2. Add ChromeDriver path in system
3. Run the test script from src folder
